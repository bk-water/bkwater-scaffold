# KoabsScaffold
KoabsScaffold 基于Spring Boot 构建

### 核心功能:

1. 通用Dao,Service,Controller抽象
2. 简单的分页功能
3. 基础的组织机构模型
4. 站内消息推送
5. 权限控制
6. 工作流

### 技术选型:

1. 基于Maven管理.
2. Mybatis
3. Spring
4. SpringMvc
5. Activiti
6. Shiro
7. Servlet3.0
8. websocket
9. Mysql


### 备注：
本项目暂时只支持Mysql数据库, 需要支持websocket 的容器(如Tomcat8)

MAVEN依赖查找： http://mvnrepository.com/artifact/org.springframework/spring-webmvc/4.1.7.RELEASE#mvn
